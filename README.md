
# Project Influencer

Bu proje pInfluencer'ın ön yüzünü içerir. Svelte-kit kullanılarak hazırlanmıştır.



## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/eminuckan/influencer
```

Proje dizinine gidin

```bash
  cd my-project
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm run dev
```
